# Image Gallery

A professional, premium responsive image gallery showcasing breathtaking photography. Built for **Codealpha Task 1**.

## Features

- **Responsive Design**: Adapts perfectly to any screen size (desktop, tablet, mobile).
- **Aesthetic UI/UX**: Uses modern CSS variables, variables, and dark/light modes.
- **Search Functionality**: Easily filter and find images.
- **Dynamic Animations**: Smooth transitions and hover effects for a premium feel.

## Technologies Used

- HTML5
- CSS3 (Vanilla CSS, Modern Layouts like Grid/Flexbox)
- JavaScript

## Setup & Run

To run this project locally, you can use any live server extension or simply open `index.html` in your browser.

If you are using Node.js:
1. Clone the repository.
2. Run `npm install` (if there are any dependencies).
3. Run `npm run dev` to start the local development server.
